🌿 Lesson 4 — Display and Elements

Display Property

Allows you to control how an element is displayed.

Types of Elements

Block-level Elements - Vertical
Examples: <p>, <ol>, <ul>, <dl>, <h1> to <h6>, <article>, <section>, <div>

Inline-level Elements - Horizontal
Examples: <a>, <span>, <strong>, <em>, <b>, <i>, <q>, <mark>

Property for Display

  • display:inline - Inline display or horizontal layout
  • display:block - Block display or vertical layout
  • display:list-item - Display as a list item <li>
  • display:none - No box or the element has no effect on the layout
  • display:inherit - Inherit value from parent

CLASS

It is a user-defined selector defined by a dot (.). You can control individual HTML element formatting with the use of classes.

Methods for writing class:

  • Indicating the HTML element followed by a dot then followed by the class name.
  • Writing the class name preceded by the dot.

CLASS SELECTOR

A style applied to a particular set of elements.

ID SELECTOR

A style applied to one element in a page. Defined by the “#” symbol. Uses the id attribute of the HTML element.

MULTIPLE SELECTORS

Two or more selectors have the same property and values, a comma is used to separate the selectors and use only one set of properties and value. No need to rewrite the whole list of properties and values.

GROUPING SELECTORS

You can group selectors and combine them in one CSS file. Each selector should be separated with a comma.
