    <div class="content">
        <p>
            In Lesson 3, I dived deeper into the core characteristics of JavaScript, learning that it is an <span class="highlight">Interpreted Language</span> that relies on the browser to execute, and a <span class="highlight">Scripting Language</span> designed to perform tasks by interacting with other elements. A major takeaway was understanding the difference between <span class="highlight">Client-Side Scripting</span>, which executes on the user's machine, and server-side scripting. This shift in location is what makes websites so responsive today.
        </p>

        <p>
            The most significant part of this lesson was mastering the <span class="highlight">Object-Oriented Programming (OOP)</span> nature of JavaScript. I learned that almost everything in the browser is an object—from the window itself to the date and time. By using <span class="highlight">Methods</span> (actions objects perform) and <span class="highlight">Properties</span> (subsections of an object), I can manipulate elements with precision. Concepts like <span class="highlight">Events</span>, which trigger code based on user interaction (like clicking or loading), and <span class="highlight">Parameters</span> have helped me understand how to build truly interactive features.
        </p>

        <p>
            Finally, I explored the practical side of JS objects like <span class="highlight">Window, Document, Math, and Navigator</span>. These objects provide the tools to do everything from opening new windows and performing complex math to identifying which browser a visitor is using. I also learned about <span class="highlight">Parsing</span>, the process where the browser converts raw text into a format the JavaScript engine can actually execute. This lesson solidified my understanding that JavaScript is not just a language; it is a powerful bridge that connects existing components to create a seamless, dynamic user experience.
        </p>
