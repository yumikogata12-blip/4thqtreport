🌸 Lesson 1 — What is CSS?

Definition

CSS (Cascading Style Sheet) describes how HTML elements are to be displayed on media. It is used to define styles for your web pages and saves a lot of work. External stylesheets are stored in CSS files.

Prerequisite

You should have a basic knowledge and understanding of HTML or XHTML to continue learning CSS.

Layers of the Webpage

  • Content (HTML): Defines the text, images, video, and everything to put in a web page using tags.
  • Presentation (CSS): Defines how the page looks in a browser, including layout, color, and fonts.
  • Behavior (JavaScript): Handles interactivity and dynamic content for the page.

HTML vs CSS

HTML: creates the content/foundation of the page.

CSS: responsible for the design, style, layout, visual effects, and colors.

Benefits of Using CSS

  • Saves work
  • Easy maintenance
  • Improves visual appeal
