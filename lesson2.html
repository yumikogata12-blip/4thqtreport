Style Sheet and Its Parts

A set of instructions to a Web browser on how to display various elements on a Web page is known as a style sheet. Every CSS is a series of instructions called statements. Identifies the elements in an HTML document that it affects and tells the browser how to draw these elements. Elements are paragraphs, links, list items, and more. Style sheets are text files or embedded content that help separate content from appearance and suggest how pages should be displayed.

How CSS is Applied

  • Linked method - CSS is stored in a separate file.
  • Embedded method - CSS is stored in the header.
  • Inline method - CSS is stored in style attributes.
