🌼 Lesson 3 — CSS Style Guide

CSS and Browsers

HTML browsers that can handle style sheets are not case-sensitive.

Punctuation in CSS

All the punctuation in CSS is important. All style declarations must have a colon between the name of the style property and the value or values assigned.

White Space

White space must exist where indicated. The user can, however, add extra whitespace to make the code more readable.

Style Properties

  • Fonts
  • Color and backgrounds
  • Text
  • Boxes and layout
  • Lists
  • Tag classification

Property Values

Keywords: They express an attribute or dimension with a reserved word. Example: xx-large, sans-serif, underline.

Length Values: They explicitly state the size of a property. They must include a two-character abbreviation of the units of measure as a suffix.

Percentage & URL

Percentage values describe proportions using % sign. URLs are relative when style sheet is external.

Color Values

Examples: red (name), #8ACE00 (hex), RGB (0%, 100%, 50%).

Multiple Style Sheets

When multiple styles exist, the last declared one takes priority.

Three CSS Style Methods

  • Linked - Using external .css file.
  • Internal - Using <style> in head.
  • Inline - Using style attribute directly.

Cascading Order

  1. Inline style
  2. External & internal styles
  3. Browser default

CSS Selectors

State which element to style.

Element Selector - Targets elements like p, h1, div.

ID Selector - Uses #name.

Class Selector - Uses .classname.

Universal Selector - Uses *

Group Selector - Uses commas.

Attribute Selector - Targets by attribute.
