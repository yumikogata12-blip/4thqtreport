<section class="reflection-section">
<p>This lesson helped me understand how CSS can be used to control the size and visibility of elements on a webpage. I learned about properties like <code>width</code>, <code>height</code>, <code>line-height</code>, and how they can be used to set the dimensions of headings, paragraphs, and other HTML elements.</p>
<p>I also discovered the <code>visibility</code> property, which can hide an element without removing it from the layout. This is useful when I want something to disappear visually but still take up space in the design.</p>
<p>One of the most interesting parts of this lesson was learning about <strong>pseudo-classes</strong>. These allow us to style elements based on their state, such as when a link is hovered over or clicked. This makes websites more interactive and user-friendly. I now better understand how styling and behavior can work together to create responsive and engaging web designs.</p>
