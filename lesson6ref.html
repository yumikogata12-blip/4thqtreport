Lesson 2 — Reflection

In this lesson, I learned that CSS is really helpful for managing the dimensions of HTML elements. I can set the width and height of different parts of a webpage, and I can also control whether something is visible or hidden using visibility. This makes it easier to organize content and make the webpage look clean and neat.

I tried applying these settings in my own example, and it was interesting to see how changing the size of one element affected the overall layout. I realized that even small adjustments in CSS can make a big difference in how a webpage looks and feels. Learning this makes me more confident in designing webpages that are both functional and visually appealing.
